# <a name="shader"></a>Шейдера

![Шейдера](images/UX/UX_Hero_StandardShader.jpg)

Поскольку holographic объектов вместе с физическими в среде, важно обеспечить визуальные подсказки в смешанной реальности. Стандартный шейдер МРТК предоставляет различные типы визуальных эффектов, которые можно использовать с голограммами. Стандартная система затенения МРТК использует один гибкий шейдер, который может обеспечить визуализацию визуальных элементов, аналогичных стандартному шейдеру Unity, реализует основные [принципы разработки](https://www.microsoft.com/design/fluent/#/)и оставаться на устройствах смешанной реальности.
