Windows Device Portal для HoloLens позволяет настраивать и управлять устройством удаленно через Wi-Fi или USB-устройство. Портал устройств — это веб-сервер на вашем устройстве HoloLens, к которому можно подключаться из веб-браузера на ПК. Портал устройств включает в себя множество средств, которые помогут вам управлять HoloLens и отладки и оптимизации приложений.

Данный документ предназначен специально о Windows Device Portal для HoloLens. С помощью портала устройства Windows для настольного компьютера (в том числе для смешанной реальности Windows), см. в разделе [Обзор Windows Device Portal](https://docs.microsoft.com/windows/uwp/debug-test-perf/device-portal)

## <a name="setting-up-hololens-to-use-windows-device-portal"></a>Настройка HoloLens для использования Windows Device Portal

1. Включите устройство HoloLens и наденьте его.
2. Выполните жест [цветение](gestures.md#bloom), чтобы запустить главное меню.
3. При помощи **параметры** плитку и выполнять [жест касания](gestures.md#air-tap) жест. Выполните второй жест касания для размещения приложения в вашей среде. Приложение «Параметры» запустится после его размещения.
4. Выберите пункт меню **Обновить**.
5. Выберите пункт меню **Для разработчиков**.
6. Включите **Режим разработчика**.
7. [Прокрутите вниз](gestures.md#composite-gestures) и включить **портал устройств**.
8. Если вы настраиваете Windows Device Portal, поэтому вы можете развертывать приложения для этого HoloLens через USB или Wi-Fi, нажмите кнопку **пары** для [создать связывания ПИН-код](using-visual-studio.md#pairing-your-device-hololens). Выйти из приложения параметров во всплывающее окно ПИН-код, пока не будет введено ПИН-кода в Visual Studio во время первого развертывания.

   ![Включение режима разработчика в настройках приложения для Windows Holographic](images/deviceportalsettings.png)

## <a name="connecting-over-wi-fi"></a>Подключение через Wi-Fi

1. [Подключение к Wi-Fi к HoloLens](connecting-to-wi-fi-on-hololens.md).
2. Поиск IP-адрес вашего устройства.
   * Найти IP-адрес на устройстве под **параметры > сеть и Интернет > Wi-Fi > Дополнительно**.
3. В веб-браузере на Компьютере перейдите к https://<YOUR_HOLOLENS_IP_ADDRESS>
   * В браузере отображается следующее сообщение: «Возникла проблема с сертификатом безопасности этого веб-сайта». Это происходит потому, что сертификат, выданный Порталу устройства, является проверочным. Сейчас можно игнорировать эту ошибку сертификата и продолжить работу.

## <a name="connecting-over-usb"></a>Подключение через USB

1. [Установите средства](install-the-tools.md) чтобы убедиться, что у вас есть Visual Studio с обновлением 1 с помощью средств разработчика Windows 10, установленными на вашем Компьютере. Это обеспечит подключение по USB.
2. Подключите HoloLens к вашему ПК с помощью кабеля micro-USB.
3. В веб-браузере вашего ПК откройте http://127.0.0.1:10080.

Щелкните этот значок: ![Значок открытия портал устройств](images/emulator-deviceportal.png) **открыть портал устройства**: Откройте Windows Device Portal для HoloLens операционной системы в эмуляторе.

## <a name="creating-a-username-and-password"></a>Создание имени пользователя и пароля

![Настройка доступа к Windows Device Portal](images/windows-device-portal-credentials-reset-page-1000px.png)<br>
*Настройка доступа к Windows Device Portal*

При первом подключении к Порталу устройства на устройстве HoloLens необходимо создать имя пользователя и пароль.
1. В веб-браузере на компьютере введите IP-адрес HoloLens. Появится страница доступа «Настройка».
2. Щелкните или коснитесь **ПИН-кода запроса** и взглянуть на экран HoloLens, чтобы получить созданный ПИН-код.
3. Введите ПИН-код в **ПИН-код показан на вашем устройстве** текстового поля.
4. Введите имя пользователя, которое будет использоваться для подключению к Порталу устройства. Это не должно быть именем учетной записи Майкрософт (MSA) или доменным именем.
5. Введите пароль и подтвердите его. Длина пароля не должна быть менее 7 символов. Он может не соответствовать паролю учетной записи MSA или домена.софт или домена.
6. Нажмите кнопку **пары** для подключения к Windows Device Portal на HoloLens.

Если вы хотите изменить это имя пользователя или пароль в любое время, можно повторить этот процесс, посетив страницу безопасности устройства, либо нажав **безопасности** ссылку в верхней правой или переходе по адресу: https://<YOUR_HOLOLENS_IP_ АДРЕС > / devicesecurity.htm.

## <a name="security-certificate"></a>Сертификат безопасности

Если вы видите в браузере ошибку сертификата, можно исправить ее, создав доверительное отношение с устройством.

Каждое устройство HoloLens создает уникальный самозаверяющий сертификат для своего подключения SSL. По умолчанию веб-браузер на вашем ПК не доверяет этому сертификату, и вы можете получить «ошибку сертификата». Скачав этот сертификат с вашего устройства HoloLens (по USB или доверенной сети Wi-Fi) и настроив доверие к нему на своем ПК, можно безопасно подключаться к устройству.
1. **Убедитесь, что вы находитесь в защищенной сети (USB или сети Wi-Fi, в которой вы доверяете).**
2. Скачайте сертификат, это устройство, на странице «Безопасность» на портале устройства.
   * Либо нажмите кнопку **безопасности** ссылку в верхнем правом списке значков или перейдите к: https://<YOUR_HOLOLENS_IP_ADDRESS>/devicesecurity.htm
3. Установите сертификат в «доверенные корневые центры сертификации» хранить на Компьютере.
   * В меню Windows введите: Управление сертификатами компьютеров и запустить приложение.
   * Разверните **доверенным корневым центром сертификации** папки.
   * Щелкните **сертификаты** папки.
   * В меню "действие" выберите: Все задачи > Импорт...
   * Выполните мастер импорта сертификатов, используя файл сертификата, загруженные с Портала устройств.
4. Перезапустите браузер.

## <a name="device-portal-pages"></a>Страницы Портала устройств

### <a name="home"></a>Корневая

![Домашняя страница Windows Device Portal на Microsoft HoloLens](images/windows-device-portal-home-page-1000px.png)<br>
*Домашняя страница Windows Device Portal на Microsoft HoloLens*

Ваш сеанс на портале устройств начинается на Главной странице. Откройте другие страницы на панели навигации вдоль левого края главной страницы.

Панель инструментов в верхней части страницы предоставляет доступ к часто используемому состоянию и функциям.
* **Online**: Указывает, подключено ли устройство к Wi-Fi.
* **Завершение работы**: Отключение устройства.
* **Перезапустите**: Циклов питания на устройстве.
* **Безопасность**: Откроется страница безопасности устройств.
* **"Холодный"**: Указывает температуру устройства.
* **ПИТАНИЯ**: Указывает, подключено ли устройство и взимать плату.
* **Справка**: Открывает страницу документации интерфейса REST.

На главной странице отображаются следующие сведения:
* **Состояние устройства:** отслеживает работоспособность устройства и сообщает о критических ошибках.
* **Сведения о Windows:** показывает имя HoloLens и текущей установленной версии Windows.
* Раздел **Настройки** содержит следующие параметры:
   * **IPD**: Расстояние interpupillary (IPD), которая является расстоянием, в миллиметрах между центре учеников пользователя при ожидании непосредственно. Этот параметр незамедлительно вступает в силу. Значение по умолчанию было вычислено автоматически при настройке устройства.
   * **Имя устройства**: Присвоить имя HoloLens. Необходимо перезапустить устройство после изменения этого параметра. После нажатия кнопки **Сохранить**, диалоговое окно запрашивает, если вы хотите перезагрузить устройство немедленно или перезагрузить позже.
   * **Параметры режимов сна**: Задает продолжительность времени ожидания, прежде чем устройство переходит в спящий режим при питании и о ситуациях от батареи.

### <a name="3d-view"></a>Трехмерное представление

![3D страница представления в Windows Device Portal на Microsoft HoloLens](images/3dview-1000px.png)<br>
*3D страница представления в Windows Device Portal на Microsoft HoloLens*

Используйте страницу «Трехмерное представление», чтобы оценить, как HoloLens воспринимает ваше окружение. В этом представлении можно перемещаться с помощью мыши:
* Сменить: щелчок левой кнопкой + мыши;
* PAN: правой кнопкой мыши, нажмите кнопку "+" мыши;
* Увеличение: прокрутки мыши.
* **Параметры отслеживания**
   * Включение непрерывного visual отслеживания путем проверки **visual отслеживания Force**. 
   * **Приостановка** останавливает visual отслеживания.
* **Просмотр параметров**: Задание параметров на объемного вида:
  * **Отслеживание**: Указывает, активен ли visual отслеживания.
  * **Показать Основание**: Отображает клетчатого floor плоскость.
  * **Показать пирамиды обзора**: Отображает усеченная.
  * **Показать плоскости стабилизации**: Отображает плоскость, которая использует HoloLens для стабилизации движения.
  * **Показать сетку**: Отображает сетку пространственное сопоставление, представляющий обстановке.
  * **Показать пространственных привязки**: Отображает Пространственные привязки для активного приложения. Необходимо нажать кнопку кнопкой "Обновить", чтобы получать и обновлять привязки.
  * **Показать подробности**: Отображает стрелки позиций, кватернионов головной поворота и вектор исходного устройства, при смене в режиме реального времени.
  * **Во весь экран кнопку**: Показывает объемного вида в полноэкранном режиме. Нажмите клавишу ESC, чтобы выйти из полноэкранного режима.
* **Область реконструкции**: Щелкните или коснитесь **обновления** для отображения сетки последние пространственное сопоставление с устройства. Выполнение полного прохода может занять некоторое время: до нескольких секунд. Сетки не обновляется автоматически в объемного вида и вручную, необходимо нажать кнопку **обновление** для получения последних сетки с устройства. Нажмите кнопку **Сохранить** для сохранения текущей сетке пространственное сопоставление в качестве obj-файл на Компьютере.
* **Пространственные привязки**: Нажмите кнопку обновления для отображения или обновления пространственных привязки для активного приложения.

### <a name="mixed-reality-capture"></a>Смешанный захват реальности

![Смешанная страница захвата реальность в Windows Device Portal на Microsoft HoloLens](images/windows-device-portal-mixed-reality-capture-page-1000px.png)<br>
*Смешанная страница захвата реальность в Windows Device Portal на Microsoft HoloLens*

Используйте [смешанной реальности захвата](mixed-reality-capture.md) страницы, чтобы сохранить потоков мультимедиа из HoloLens.
* **Параметры**: Управления мультимедиа, которые регистрируются, установив следующие параметры:
  * **Голограммы**: Захватывает holographic содержимое в поток видео. Голограммы обрабатываются в режиме моно, а не стерео.
  * **Камера PV**: Записывает поток видео от камеры, фото и видео.
  * **MIC аудио**: Записывает звук из массива "микрофон".
  * **Приложение аудио**: Записывает звук из запущенного приложения.
  * **Качество предварительной версии Live**: Выберите разрешение экрана, частота кадров и частота потоковой передачи для динамического просмотра.
* Щелкните или коснитесь **динамического просмотра** кнопку, чтобы показать поток записи. **Динамический просмотр Stop** останавливает поток записи.
* Щелкните или коснитесь **записи** для запуска записи потока смешанной реальности, используя указанные параметры. **Остановить запись** завершает запись и сохраняет его.
* Щелкните или коснитесь **Take photo** вступили в образ будет по-прежнему из потока записи.
* **Видео и фотографии**: Список видео и фотографий выделений, затраченное на устройстве.

Обратите внимание, что приложения HoloLens не смогут захватить фотографию или видео MRC, пока вы записываете или воспроизводите поток в режиме предварительного просмотра в реальном времени на Портале устройств.

### <a name="performance-tracing"></a>Мониторинг производительности

Чтобы остановить трассировку, щелкните ссылку остановки. Остаться на этой странице, до завершения загрузки файла трассировки.

Захваченные файлы ETL можно открыть для анализа в [Windows Performance Analyzer](https://msdn.microsoft.com/library/windows/hardware/hh448170.aspx).

### <a name="system-performance"></a>Производительность системы

*Страница производительности системы в Windows Device Portal на Microsoft HoloLens*

* **ОПЕРАЦИЙ ВВОДА-ВЫВОДА**: Операции чтения и записи
* **Сеть**: Получаемых и отправляемых
* **Память**: Всего используется, зафиксирована, разбитых на страницы и невыгружаемого

### <a name="logging"></a>Ведение журнала

![Страница «Регистрация» в Windows Device Portal на Microsoft HoloLens](images/windows-device-portal-logging-page-1000px.png)<br>
*Страница «Регистрация» в Windows Device Portal на Microsoft HoloLens*

Управление в реальном времени событий трассировки для Windows (ETW) на HoloLens.

Проверьте **скрыть поставщики** Показать **события** только список.
* **Зарегистрированных поставщиков**: Выберите поставщика трассировки событий Windows и уровень трассировки. Уровень трассировки — это одно из следующих значений:
   1. ненормальный выход или прекращение;
   2. серьезные ошибки;
   3. Предупреждения
   4. предупреждения, не связанные с ошибками.

Щелкните или нажмите **Включить** для запуска трассировки. Поставщик добавляется в раскрывающийся список **Включенные поставщики**.
* **Настраиваемые поставщики**: Выберите пользовательский поставщик трассировки событий Windows и уровень трассировки. Определите поставщика по его GUID. Не включайте скобки в GUID.
* **Во включенных регистраторах**: Список разрешенных поставщиков. Выберите поставщика в раскрывающемся списке и щелкните или нажмите **Отключить**, чтобы остановить трассировки. Щелкните или нажмите **Стоп**, чтобы приостановить всю трассировку.
* **Журнал поставщиков**: Представлены поставщики трассировки событий Windows, которые были включены в текущем сеансе. Щелкните или нажмите **Включить**, чтобы активировать поставщика, который ранее был отключен. Щелкните или нажмите **Очистить**, чтобы очистить журнал.
* **события**: Список событий трассировки событий Windows из выбранных поставщиков в табличном формате. Эта таблица обновляется в режиме реального времени. Под таблицей, щелкните **Очистить** кнопку, чтобы удалить все события трассировки событий Windows из таблицы. Это не приводит к отключению каких-либо поставщиков. Вы можете щелкнуть **Сохранить в файл**, чтобы экспортировать собранные события трассировки событий Windows локально в CSV-файл.
* **Фильтры**: Позволяют фильтровать события трассировки событий Windows, собранные системой идентификатор, ключевое слово, уровень, имя поставщика, имя задачи или текст. Условия можно комбинировать:
   1. Для применения условия к тому же свойству - события может удовлетворять отображаются любого из этих критериев.
   2. Для критериев применение различных свойству - событий должен удовлетворять всем условиям

Например, можно указать критерии *(имя задачи содержит «Foo» или «Панели») и (текст содержит «error» или «предупреждение»)*

### <a name="simulation"></a>Simulation

![Страница моделирование в Windows Device Portal на Microsoft HoloLens](images/windows-device-portal-simulation-page-1000px.png)<br>
*Страница моделирование в Windows Device Portal на Microsoft HoloLens*

Позволяет записывать и воспроизводить входные данные для тестирования.
* **Запись места**: Используется для загрузки файла имитации места, содержащего пространственное сопоставление сети для окружения пользователя. Имя комнаты и нажмите кнопку **захвата** для сохранения данных в формате .xef на вашем Компьютере. Этот файл комнаты может быть загружен в эмулятор HoloLens.
* **Запись**: Проверка потоков для записи, введите имя записи и щелкните или коснитесь **записи** чтобы начать запись. Выполнение действий с вашей HoloLens и нажмите кнопку **остановить** для сохранения данных в формате .xef на вашем Компьютере. Этот файл может быть загружен в эмулятор или на устройство HoloLens.
* **Воспроизведение**: Щелкните или коснитесь **отправки записи** для выбора файла xef с вашего компьютера и отправить данные в HoloLens.
* **Режим управления**: Выберите **по умолчанию** или **моделирования** из раскрывающегося списка и нажмите кнопку **задать** кнопку, чтобы выбрать режим на HoloLens. Выбор «Симуляторы» отключает датчики на устройстве HoloLens и использует вместо них переданные смоделированные данные. Если переключиться на «Симуляторы», устройство HoloLens не будет реагировать на пользователя, пока не переключитесь обратно на значение «По умолчанию».
