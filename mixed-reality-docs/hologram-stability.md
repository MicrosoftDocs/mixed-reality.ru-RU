* <span data-ttu-id="421bd-306">Объекты, которые быстро перемещаются, включая объекты, заблокированные в голову, например [курсор](cursors.md).</span><span class="sxs-lookup"><span data-stu-id="421bd-306">Objects that are moving quickly, including head-locked objects such as the [cursor](cursors.md).</span></span>
* <span data-ttu-id="421bd-307">Объекты, которые значительно далеко от [плоскости стабилизации](hologram-stability.md#reprojection).</span><span class="sxs-lookup"><span data-stu-id="421bd-307">Objects that are substantially far from the [stabilization plane](hologram-stability.md#reprojection).</span></span>

<span data-ttu-id="421bd-308">**Для ослабления эффекта разделения цветов:**</span><span class="sxs-lookup"><span data-stu-id="421bd-308">**To attenuate the effects of color-separation:**</span></span>
* <span data-ttu-id="421bd-309">Сделайте объект задержкой пользователя.</span><span class="sxs-lookup"><span data-stu-id="421bd-309">Make the object lag the user's gaze.</span></span> <span data-ttu-id="421bd-310">Он должен выглядеть так, как если бы он был в некоторой инерции и был прикреплен к взгляду «по пружины».</span><span class="sxs-lookup"><span data-stu-id="421bd-310">It should appear as if it has some inertia and is attached to the gaze "on springs".</span></span> <span data-ttu-id="421bd-311">Это замедлит работу курсора (уменьшая расстояние разделения) и помещает его за наиболее вероятную точку.</span><span class="sxs-lookup"><span data-stu-id="421bd-311">This slows the cursor (reducing separation distance) and puts it behind the user's likely gaze point.</span></span> <span data-ttu-id="421bd-312">Пока он быстро перехватывается, когда пользователь останавливает сдвиг своего взгляда, он выглядит довольно естественным.</span><span class="sxs-lookup"><span data-stu-id="421bd-312">So long as it quickly catches up when the user stops shifting their gaze it feels quite natural.</span></span>
* <span data-ttu-id="421bd-313">Если вы хотите переместить голограмму, постарайтесь, чтобы скорость перемещения была меньше 5 градусов в секунду, если предполагается, что пользователь будет следовать за ним.</span><span class="sxs-lookup"><span data-stu-id="421bd-313">If you do want to move a hologram, try to keep it's movement speed below 5 degrees/second if you anticipate that the user will follow it with their eyes.</span></span>
* <span data-ttu-id="421bd-314">Для курсора используйте *светлое* вместо *Geometry* .</span><span class="sxs-lookup"><span data-stu-id="421bd-314">Use *light* instead of *geometry* for the cursor.</span></span> <span data-ttu-id="421bd-315">Источник виртуального освещения, присоединенный к указателю, будет рассматриваться как интерактивный указатель, но не будет приводить к разделению цветов.</span><span class="sxs-lookup"><span data-stu-id="421bd-315">A source of virtual illumination attached to the gaze will be perceived as an interactive pointer but will not cause color-separation.</span></span>
* <span data-ttu-id="421bd-316">Настройте плоскость стабилизации в соответствии с голограммами, на которых пользователь облаками.</span><span class="sxs-lookup"><span data-stu-id="421bd-316">Adjust the stabilization plane to match the holograms the user is gazing at.</span></span>
* <span data-ttu-id="421bd-317">Сделайте объект красным, зеленым или синим.</span><span class="sxs-lookup"><span data-stu-id="421bd-317">Make the object red, green or blue.</span></span>
* <span data-ttu-id="421bd-318">Переключиться на размытую версию содержимого.</span><span class="sxs-lookup"><span data-stu-id="421bd-318">Switch to a blurred version of the content.</span></span> <span data-ttu-id="421bd-319">Например, круглый белый курсор может быть изменен на немного размытую линию, ориентированную на направление движения.</span><span class="sxs-lookup"><span data-stu-id="421bd-319">For example, a round white cursor could be change to a slightly blurred line oriented in the direction of motion.</span></span>

<span data-ttu-id="421bd-320">Как и раньше, отрисовка в 60 кадров/с и настройка плоскости стабилизации является наиболее важными методиками для повышения стабильности.</span><span class="sxs-lookup"><span data-stu-id="421bd-320">As before, rendering at 60 FPS and setting the stabilization plane are the most important techniques for hologram stability.</span></span> <span data-ttu-id="421bd-321">При заметном разделении цветов сначала убедитесь, что частота кадров соответствует ожиданиям.</span><span class="sxs-lookup"><span data-stu-id="421bd-321">If facing noticeable color separation, first make sure the frame rate meets expectations.</span></span>
