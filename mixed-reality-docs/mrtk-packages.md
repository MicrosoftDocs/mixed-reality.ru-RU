# <a name="mrtk-packages"></a>Пакеты MRTK

Набор средств смешанной реальности (MRTK) является коллекцией из пакетов, обеспечивающих разработку приложений межплатформенные смешанной реальности, предоставляя поддержку для смешанной реальности оборудования и платформ компонентной образом.

Существует три категории пакетов MRTK: 

* [Foundation](#foundation-packages)
* [Расширение](#extension-packages)
* [Экспериментальные](#experimental-packages)

## <a name="foundation-packages"></a>Базовые пакеты

В смешанной реальности Toolkit лежит набор пакетов, которые позволяют приложению использовать функциональные возможности на платформах смешанной реальности. Эти пакеты будут выпущены и поддерживается корпорацией Майкрософт, из исходного кода в [mrtk_release](https://github.com/Microsoft/MixedRealityToolkit-Unity/tree/mrtk_release) ветви на сайте GitHub.

![MRTK базовые пакеты](images/mrtk-foundation.jpg)

*Смешанной реальности Toolkit базовые пакеты*

MRTK Foundation состоит из:

* [Основной пакет](#core-package)
* [Поставщики платформы](#platform-providers)
* [Системные службы](#system-services)
* [Функция активы](#feature-assets)

В следующих разделах описаны типы пакетов в каждой категории.

### <a name="core-package"></a>Основной пакет

Основной пакет является _требуется_ компонента и выполняется как зависимость использоваться всеми пакетами MRTK foundation.

MRTK основной пакет включает в себя:

* [Общие интерфейсы, классы и типы данных](#common-interfaces-clases-and-data-types)
* [Компонент MixedRealityToolkit сцены](#mixedrealitytoolkit-scene-component)
* [Стандартный шейдера MRTK](#mrtk-standard-shader)
* [Поставщик ввода с Unity](#unity-input-provider)
* [Управление пакетами](#package-management)

#### <a name="common-interfaces-classes-and-data-types"></a>Общие интерфейсы, классы и типы данных

Основной набор средств смешанной реальности пакет содержит определения для всех общих интерфейсов, классов и типов данных, которые используются другими компонентами. Настоятельно рекомендуется что приложений доступа к компонентам MRTK исключительно через определенные интерфейсы, позволяющие самый высокий уровень совместимости между платформами.

#### <a name="mixedrealitytoolkit-scene-component"></a>Компонент MixedRealityToolkit сцены

Компонент сцены MixedRealityToolkit — диспетчер централизованного отдельный ресурс для смешанной реальности Toolkit. Этот компонент загружает и управляет существования модулей платформ и служб и предоставляет ресурсы для систем, для доступа к их параметры конфигурации. 

#### <a name="mrtk-standard-shader"></a>Стандартный шейдера MRTK

Стандартный шейдера MRTK обеспечивает основу для практически всех материалов, предоставленных MRTK. Этот шейдер чрезвычайно гибкого и оптимизированного для различных платформ, в которых поддерживается MRTK. Это _высокой_ рекомендуется использовать стандартный шейдера MRTK материалы приложения для обеспечения оптимальной производительности.

#### <a name="unity-input-provider"></a>Поставщик ввода с Unity

Поставщик ввода Unity предоставляет доступ к common устройства ввода, например игровых устройств управления, сенсорных экранов и 3D пространственных мыши.

#### <a name="package-management"></a>Управление пакетами

_Скоро_

Основной набор средств смешанной реальности пакет предоставляет поддержку обнаружения и управления необязательно foundation, расширение и экспериментальные пакеты MRTK.

### <a name="platform-providers"></a>Поставщики платформы

Пакеты MRTK поставщика платформы являются компонентами, позволяющих смешанной реальности Toolkit к целевому объекту смешанной реальности оборудования и функциональных возможностей платформы.

Поддерживаются следующие платформы:

* [Windows Mixed Reality](#windows-mixed-reality)
* [OpenVR](#openvr)
* [Голосовые Windows](#windows-voice)

#### <a name="windows-mixed-reality"></a>Windows Mixed Reality

Пакет Windows Mixed Reality обеспечивает поддержку для устройств Microsoft HoloLens и Иммерсивных Windows смешанной реальности. Пакет содержит полной поддержки платформы, включая:

* Нацеливание взглядом
* Жесты
* Контроллеры Windows смешанной реальности движения
* Пространственное сопоставление

#### <a name="openvr"></a>OpenVR

Пакет OpenVR обеспечивает поддержку оборудования и платформы для устройств с помощью платформы OpenVR.

#### <a name="windows-voice"></a>Голосовые Windows

Пакет Windows голосовой обеспечивает поддержку функций распознавания ключевое слово, на устройствах Microsoft Windows 10.

### <a name="system-services"></a>Системные службы

Базовые службы платформы, указанные в пакеты службы системы. Эти пакеты содержат системные службы интерфейсы, определенные в реализации по умолчанию смешанной реальности Toolkit [core](#core-package) пакета.

MRTK foundation включает в себя следующие системные службы:

* [Границы системы](#boundary-system)
* [Система диагностики](#diagnostic-system)
* [Система ввода](#input-system)
* [Поддержка пространственных системы](#spatial-awareness-system)
* [Teleport системы](#teleport-system)

#### <a name="boundary-system"></a>Границы системы

Система границ MRTK предоставляет данные о в виртуальный реальности playspace. В системах, для которых пользователь настроил границу система может предоставлять плоскость floor, прямоугольная playspace, отслеживаемой области и многое другое.

#### <a name="diagnostic-system"></a>Система диагностики

Система диагностики MRTK предоставляет данные производительности в реальном времени в работу приложения. В краткая будет иметь возможность просматривать частоту кадров, загруженности процессора и других ключевых метрик производительности, использования приложения.

#### <a name="input-system"></a>Система ввода

MRTK систем ввода позволяет приложениям получать доступ к входные данные в виде кроссплатформенный, определение действия пользователя и назначение эти действия наиболее подходящий кнопок и осей на целевых контроллеров.

#### <a name="spatial-awareness-system"></a>Поддержка пространственных системы

Пространственные системы Awareness MRTK обеспечивает доступ к реальной среды данных из устройств, таких как Microsoft HoloLens.

#### <a name="teleport-system"></a>Teleport системы

Система Teleport MRTK поддерживает locomotion виртуальной реальности.

### <a name="feature-assets"></a>Функция активы

Функция ресурсы представляют собой коллекции связанных функций, предоставляемая как Unity средств и сценариев. Ниже перечислены некоторые из этих функций.

* Стандартных активов

## <a name="extension-packages"></a>Пакеты расширений

Пакеты расширений MRTK представляют собой коллекцию пакеты, написанные корпорацией Майкрософт и сообщества, которые расширяют и улучшают функции набора средств смешанной реальности. Авторы расширение будет состояние все необходимые зависимости, пометить пакет как совместимая с набором средств смешанной реальности и предоставляют лицензирования и поддерживают инструкции.

Пакеты расширений может предоставлять новые функции и поддержку новых платформ. Со временем расширения могут с помощь и утверждения авторов, быть перенесены в Foundation MRTK, после чего они будут выпущены и поддерживается корпорацией Майкрософт.

## <a name="experimental-packages"></a>Экспериментальные пакеты

Экспериментальные пакеты предоставляют возможность рейсов прототипа функции, предварительные выпуски и интересные новые идеи. Цель наиболее экспериментальные пакеты — попробовать что-то новое или датчика заинтересованности клиентов. Многие, хотя и не все экспериментальные пакеты будут повторно выпущено как расширения по завершении создания прототипов и тестирования.

## <a name="conclusion"></a>Заключение

Пакеты смешанной реальности Toolkit и система управления пакетами предназначены для включения простой и четкий способ аддитивно создания функции в своем опыте работы, не требуя ненужные компоненты для включения в проект.
