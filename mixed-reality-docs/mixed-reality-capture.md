# <a name="mixed-reality-capture"></a>Смешанный захват реальности

HoloLens дает пользователям возможность смешивать мир в реальном мире с цифровым миром. Запись смешанной реальности (НОРМАТИВная реальность) позволяет захватить этот опыт как фотографию или видео. Это позволяет использовать взаимодействие с другими пользователями, позволяя им видеть голограммы по мере их появлении. Такие видеоролики и фотографии изменяются с точки зрения первого лица. Для точки зрения стороннего человека используйте [представление spectator](spectator-view.md).

Варианты использования для захвата смешанной реальности выходят за пределы общего доступа к видео в социальных кругах. Видео можно использовать, чтобы сообщить другим пользователям о том, как использовать приложение. Разработчики могут использовать видео или по-прежнему для улучшения возможностей воспроизведения и отладки приложений.

## <a name="live-streaming-from-hololens"></a>Потоковая трансляция из HoloLens

[Обновление Windows 10 октября 2018](release-notes-october-2018.md) г. добавляет поддержку Miracast в HoloLens. Нажмите кнопку **подключить** в нижней части меню Пуск, чтобы открыть средство выбора для устройств и адаптеров с поддержкой Miracast. Выберите устройство, на котором необходимо начать потоковую передачу. По завершении нажмите кнопку **Отключить** в нижней части меню "Пуск".  Функции **подключения** и **отключения** также доступны в меню Быстрые действия.

[Портал устройств Windows](using-the-windows-device-portal.md) и [вспомогательное приложение Microsoft HoloLens](https://www.microsoft.com/store/productId/9NBLGGH4QWNX) предоставляют динамические возможности потоковой передачи для устройств, которые находятся в режиме разработчика.

[Служба удаленного помощника Dynamics 365](https://dynamics.microsoft.com/en-us/mixed-reality/remote-assist) поддерживает динамическую потоковую передачу от HoloLens к сотрудникам в удаленных расположениях.

## <a name="taking-mixed-reality-captures"></a>Сбор данных смешанной реальности

![Щелкните значок камеры в нижней части меню "Пуск".](images/cameraiconinpins-300px.png)<br>
*Щелкните значок камеры в нижней части меню "Пуск".*

Существует несколько способов инициировать запись смешанной реальности.
* Кортана может использоваться во всех случаях независимо от выполняемого в данный момент приложения. Просто скажите: "Привет, Кортана, возьмем фотографию" или "Привет, Кортана, начало записи". Чтобы закрыть видео, скажите "Привет, Кортана, завершение записи".
* В меню Пуск выберите **Камера** или **видео**. Использование [воздушного касания](gestures.md#air-tap) для открытия встроенного пользовательского интерфейса для камеры с нормативными названиями.
* В меню Быстрые действия выберите **Камера** или **видео** , чтобы открыть встроенный пользовательский интерфейс "биокамера".
* Приложения могут предоставлять собственный пользовательский интерфейс для записи смешанной реальности с помощью пользовательского интерфейса или, начиная с [обновления Windows 10 октября 2018](release-notes-october-2018.md), [встроенного пользовательского интерфейса камеры](mixed-reality-capture-for-developers.md)с нормативными потребностями.
* Уникальный для HoloLens: 
    * На [портале устройств Windows](using-the-windows-device-portal.md) есть страница записи смешанной реальности, которую можно использовать для создания фотографий, видео, динамического потока и просмотра записей.
    * Одновременно нажмите одновременно  кнопки громкость и **понижение громкости** , чтобы сделать снимок, независимо от выполняемого в данный момент приложения.
    * Чтобы начать  запись видео, удерживайте кнопки громкости и **уменьшения** громкости в течение трех секунд. Чтобы закрыть видео, нажмите одновременно кнопки **Громкость** и **Громкость** .
* Уникальный для впечатляющих головных телефонов: 
    * С помощью контроллера движения удерживайте кнопку **Windows** , а затем коснитесь триггера  , чтобы сделать снимок. 
    * Используя контроллер движения, нажмите кнопку **Windows** , а затем нажмите кнопку **меню** , чтобы начать запись видео. Нажмите кнопку **Windows** , а затем выберите **триггер** , чтобы закончить запись видео.
    
>[!NOTE]
>[Обновление Windows 10 от октября 2018](release-notes-october-2018.md) г. изменит, как работает кнопка раскрытия и Windows. Перед обновлением раскрытия жест или нажатие кнопки Windows приведет к прерыванию записи. После обновления с помощью жеста раскрытия или кнопки Windows открывается меню "Пуск" (или меню "быстрые действия", если вы используете приложение). В меню выберите команду " **Закрыть видео** ", чтобы отменить запись.

### <a name="limitations-of-mixed-reality-capture"></a>Ограничения записи смешанной реальности

В HoloLens система будет регулировать скорость рендеринга до 30Hz. Это создает некоторый запас для выполнения нормативных требований, поэтому приложению не нужно поддерживать постоянный запас бюджета, а также соответствует частоты кадров записи видео о НОРМАТИВах (до) 30fps.

Максимальная длина видео — пять минут.

Встроенный пользовательский интерфейс для камеры с частотой требований поддерживает только одну операцию с соблюдением нормативных требований (создание изображения является взаимоисключающим при записи видео).

### <a name="file-formats"></a>Форматы файлов

Смешанная реальность захватывается голосовыми командами Кортаны и инструментами меню "Пуск" создает файлы в следующих форматах:

|  Type  |  Формат  |  Расширение  |  Разрешение  |  Звук | 
|----------|----------|----------|----------|----------|
|  Photo  |  [РИСУНОК](https://en.wikipedia.org/wiki/JPEG)  |  .jpg  |  3904x2196px (HoloLens 2)<br> 1408x792px (HoloLens)<br> 1920x1080px (иммерсивное гарнитуры) |  Н/Д | 
|  Видео  |  [MPEG-4](https://en.wikipedia.org/wiki/MPEG-4)  |  .mp4  |  1920x1080px на 30fps (HoloLens 2)<br> 1216x684px на 24fps (HoloLens)<br> 1632x918px в 30fps (впечатляющие головные телефоны) |  Стерео 48kHz | 

>[!NOTE]
>Разрешение фотографий и видео может быть небольшим, если фото-или видеокамера уже используется другим приложением, в потоковой трансляции или когда системные ресурсы недостаточны.

### <a name="video-stabilization"></a>Видео стабилизации

По умолчанию:
* Видео стабилизации с нулевой задержкой применяется при динамической потоковой передаче по Miracast.
* Видео стабилизации с длительной задержкой применяется к видеороликам, захваченным с помощью встроенного пользовательского интерфейса "биокамера", команды Кортаны и портал устройств Windows.

## <a name="viewing-mixed-reality-captures"></a>Просмотр записей смешанной реальности

Фотографии и видеозаписи смешанной реальности сохраняются в папке "пленка" камеры устройства. Доступ к ним можно получить с помощью [приложения "фотографии](see-your-photos.md#photos-app) " или проводника.

На компьютере, подключенном к HoloLens, можно также использовать [портал устройств Windows](using-the-windows-device-portal.md#mixed-reality-capture) или проводник на компьютере ([через MTP](release-notes-april-2018.md#new-features-for-hololens)).

Если вы устанавливаете [приложение OneDrive](https://www.microsoft.com/p/onedrive/9wzdncrfj1p3), вы можете включить **передачу камеры,** и ваши фотографии и видео с соблюдением требований будут синхронизироваться с onedrive и другими устройствами с помощью onedrive.

>[!NOTE]
>Начиная с обновления Windows 10 апреля 2018, приложение Photos больше не будет отправлять фотографии и видео в OneDrive.
