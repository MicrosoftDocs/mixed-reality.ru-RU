## <a name="chapter-1---create-a-new-project"></a>Глава 1 - Создание нового проекта

>[!VIDEO https://www.youtube.com/embed/2L5IFO0hnYA]

Чтобы создать приложение с помощью Unity, необходимо сначала создать проект. Этот проект организована в несколько папок, наиболее важным из которых является папку ресурсы. Это папка, содержащая все ресурсы, импортированные из средства Создание цифрового содержимого, такие как Maya, кинотеатрах максимум 4D или Photoshop, весь код, созданных с помощью Visual Studio или свой основной редактор кода, и любое количество файлов содержимого, что Unity создается такое же вы объедините сцены , анимации и другие типы активов Unity в редакторе.

Для создания и развертывания приложений универсальной платформы Windows, Unity можно экспортировать как решение Visual Studio, который будет содержать все необходимые средства и файлы кода проекта.

1. Запустите Unity
2. Выберите **New**
3. Введите имя проекта (например) «MixedRealityIntroduction»)
4. Введите расположение для сохранения проекта
5. Убедитесь, **3D** выбран переключатель
6. Выберите **создать проект**

Поздравляем Вы являетесь информации о настройке быстро начать работу с настройки смешанной реальности.

Unity Main Camera обрабатывает отслеживания head "и" stereoscopic Подготовка к просмотру. Существуют некоторые изменения, чтобы вносить Main Camera, чтобы использовать его с помощью смешанной реальности.
1. Выберите Файл > создать сцену

Во-первых, он будет проще макета приложения, если представить пользователя в качестве начальной позиции (**X**: 0, **Y**: 0, **Z**: 0). Поскольку Main Camera отслеживает перемещение головы пользователя, задавая начальное положение Main Camera можно задать начальную позицию пользователь.
1. Выберите **Main Camera** в **иерархии** панели
2. В **инспектор** панели, найти **преобразования** компонента и изменение **позиции** из (**X**: 0, **Y**: 1, **Z**: -10) для (**X**: 0, **Y**: 0, **Z**: 0)

Во-вторых фон камеры по умолчанию необходимо продумать.

**Для приложений HoloLens**, реальный мир должен отображаться за все камеры подготовке к просмотру не Skybox текстуры.
1. С помощью **Main Camera** снимая выделение в **иерархии** панели, найти **камеры** компонент в **инспектор** панели и изменить **Очистить флаги** раскрывающийся список **Skybox** для **Одноцветная**.
2. Выберите **фона** палитра цветов и изменение **RGBA** значения (0, 0, 0, 0)

**Для смешанной реальности приложения, назначенные для иммерсивную**, мы можем использовать текстуру Skybox по умолчанию, среда Unity предоставляет.
1. С помощью **Main Camera** снимая выделение в **иерархии** панели, найти **камеры** компонент в **инспектор** панели и сохранить **Очистить флаги** раскрывающийся список, чтобы **Skybox**.

В-третьих давайте рассмотрим ближней отсекающей плоскости в Unity и предотвращают объектов представляется слишком близко к пользователям глаза как пользователь достигает объект или объект приближается к пользователю.

**Для приложений HoloLens**, может быть присвоено ближней отсекающей плоскости [HoloLens рекомендуется](camera-in-unity.md#clip-planes) 0,85 метров.
1. С помощью **Main Camera** снимая выделение в **иерархии** панели, найти **камеры** компонент в **инспектор** панели и изменить **Практически отсекающей плоскости** поле по умолчанию **0,3** для HoloLens рекомендуется **0,85**.

**Для смешанной реальности приложения, назначенные для иммерсивную**, можно использовать параметр по умолчанию, который предоставляет Unity.
1. С помощью **Main Camera** снимая выделение в **иерархии** панели, найти **камеры** компонент в **инспектор** панели и сохранить **Практически отсекающей плоскости** поле по умолчанию **0,3**.

Наконец сообщите нам сохраните наши достижения промежуточные. Чтобы сохранить изменения сцены, выберите **файл > Сохранить сцену как**, присвойте сцене имя **Main**и выберите **Сохранить**.

Мы также будет задать некоторые параметры качества для нашего приложения. Наконец мы гарантируем, что наших целевых объектов сборки установлены в Windows Store.

## <a name="chapter-4---create-a-cube"></a>Глава 4 — Создание куба

>[!VIDEO https://www.youtube.com/embed/qKcK1Yuj-HQ]

Создание куба в проекте Unity — аналогично созданию любого другого объекта в Unity. Размещение куба глазах у пользователя несложно, так как система координат Unity сопоставляется реальном мире - где метра в Unity — приблизительно метра в реальном мире.
1. В верхнем левом углу **иерархии** панели, выберите **создать** раскрывающийся список и выберите **трехмерный объект > куба**.
2. Выберите только что созданный **куба** в **иерархии** панели
3. В **инспектор** найти **преобразования** компонента и изменение **позиции** для (**X**: 0, **Y**: 0, **Z**: 2). *Это размещает куба 2 м перед его начальной позиции.*
4. В **преобразования** компонента, изменение **поворота** для (**X**: 45, **Y**: 45, **Z**: 45) и измените **масштабирования** для (**X**: 0,25, **Y**: 0,25, **Z**: 0.25). *При этом размер куба 0,25 метров.*
5. Чтобы сохранить изменения сцены, выберите **файл > Сохранить сцену**.

## <a name="chapter-5---verify-on-device-from-unity-editor"></a>Глава 5 - убедитесь в том, на устройстве из редактора Unity

>[!VIDEO https://www.youtube.com/embed/vmCfiIdRb6Q]

Теперь, после создания куба, настала пора выполнить быструю проверку на устройстве. Это можно сделать непосредственно из в редакторе Unity.

### <a name="initial-setup"></a>Начальная настройка
1. На Компьютере, разработки в Unity, откройте **файл > Параметры сборки** окна.
2. Изменение **платформы** для **универсальной платформы Windows** и нажмите кнопку **платформам коммутатора**

### <a name="for-hololens-use-unity-remoting"></a>Для HoloLens использования удаленного взаимодействия для Unity
1. На ваш HoloLens, установить и запустить [Holographic проигрывателя удаленного взаимодействия](holographic-remoting-player.md), доступном Windows Store. Запустите приложение на устройстве, и он будет переходить в состояние ожидания и отобразить IP-адрес устройства. Запишите IP-адрес.
2. Откройте **окна > XR > Holographic эмуляции**.
3. Изменение **режима эмуляции** из **None** для **удаленного устройство**.
4. В **удаленный компьютер**, введите IP-адрес вашего HoloLens, записанные ранее.
5. Нажмите кнопку **Подключить**.
6. Убедитесь, **состояние подключения** становится зеленым **подключено**.
7. Теперь необходимо щелкнуть **воспроизведение** в редакторе Unity.

Теперь можно видеть куба на устройстве и в редакторе. Можно приостановить, проверять объекты и отладки так же, как вы запускаете приложение в редакторе, потому что по сути это том, что происходит, но с видео, аудио и устройствами ввода, и обратно при передаче по сети между хост-компьютере и устройством.

### <a name="for-other-mixed-reality-supported-headsets"></a>Для других смешанной реальности поддерживается гарнитуры

1. Подключитесь к Компьютеру с помощью USB-кабеля и HDMI разработки гарнитуры или отображения кабель порт.
2. Запустите **смешанной реальности портала** и убедитесь, что вы выполнили первого запуска.
3. Из Unity теперь можно нажать кнопку воспроизведения.

Теперь можно увидеть отрисовку куба, в вашей гарнитуры смешанной реальности и в редакторе.

## <a name="chapter-6---build-and-deploy-to-device-from-visual-studio"></a>Глава 6 - создание и развертывание на устройстве из Visual Studio

>[!VIDEO https://www.youtube.com/embed/USSu8yHUdbk]

Теперь мы готовы для компиляции наш проект в Visual Studio и развертывания наших целевое устройство.

### <a name="export-to-the-visual-studio-solution"></a>Экспорт в решение Visual Studio
1.  Откройте **файл > Параметры сборки** окна.
2.  Нажмите кнопку **добавьте откройте сцены** Добавление сцены.
3.  Изменение **платформы** для **универсальной платформы Windows** и нажмите кнопку **переключить платформу**.
4.  В **Windows Store** параметры обеспечения **SDK** — **универсальной 10**.
5.  Для целевого устройства, оставьте **любого устройства** перекрыто отображает или переключиться в режим **HoloLens**.
6.  **Тип сборки UWP** должно быть **D3D**.
7.  **Пакет SDK для UWP** может остаться в **самую новую установленную**.
8.  Проверьте **Unity C# проекты** в режиме отладки.
9.  Щелкните **Сборка**.
10. В проводнике щелкните **новую папку** и назовите папку **«Приложение»**.
11. С помощью **приложения** щелкните папку, **Выбор папки** кнопки.
12. После завершения Unity построения, появится окно проводника Windows.
13. Откройте **приложения** папку в проводнике.
14. Откройте созданный решение Visual Studio (MixedRealityIntroduction.sln в этом примере)

### <a name="compile-the-visual-studio-solution"></a>Скомпилируйте решение Visual Studio

Наконец мы компиляции экспортированного решения Visual Studio, развернуть его и опробовать его на устройстве.
1. С помощью верхней панели инструментов в Visual Studio, изменить целевую платформу в списке **Отладка** для **выпуска** и из **ARM** для **X86**.

Инструкции зависят от развертывание на устройстве или эмуляторе. Следуйте инструкциям, которые соответствуют вашей установки.

### <a name="deploy-to-mixed-reality-device-over-wi-fi"></a>Развертывание по Wi-Fi на устройстве смешанной реальности
1. Щелкните стрелку рядом с полем **локального компьютера** кнопку и цели развертывания, чтобы изменить **удаленный компьютер**.
2. Введите IP-адрес устройства смешанной реальности и измените **режим проверки подлинности** универсальный (незашифрованный протокол) для HoloLens и **Windows** для других устройств.
3. Нажмите кнопку **Отладка > Запуск без отладки**.

**Для HoloLens**, если это при первом развертывании на устройстве, вам потребуется пара [с помощью Visual Studio](using-visual-studio.md).

### <a name="deploy-to-mixed-reality-device-over-usb"></a>Развертывание по USB на устройстве смешанной реальности

Убедитесь, что ваше устройство подключено через USB-кабеля.
1. **Для HoloLens**, щелкните стрелку рядом с полем **локального компьютера** кнопку и цели развертывания, чтобы изменить **устройства**.
2. **Для нацеливания перекрыто устройств, подключенных к ПК**, оставить значение на локальный компьютер. Убедитесь в наличии **смешанной реальности портала** под управлением.
3. Нажмите кнопку **Отладка > Запуск без отладки**.

### <a name="deploy-to-emulator"></a>Развертывание в эмуляторе
1. Щелкните стрелку рядом с полем **устройства** кнопку и в раскрывающемся списке выберите **HoloLens эмулятор**.
2. Нажмите кнопку **Отладка > Запуск без отладки**.

### <a name="try-out-your-app"></a>Проверьте работу приложения

Теперь, когда ваше приложение развертывается, попробуйте переместить по всему куба и обратите внимание, что он остается в мире перед глазами.
